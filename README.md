## How To Guide.

### Pre Requisite

- docker and docker-compose
- node.js
- nest.js cli

### Prepare Project

1.  Clone Boilerplate Project.
    ```shell
    get clone https://github.com/DrZIn-dev/nest.js_mongo_k8s.git
    ```
2.  Run Project.
    ```
    npm install
    npm run start:dev
    ```
3.  Run Database As Container.
    ```
    docker-compose up -d
    ```
